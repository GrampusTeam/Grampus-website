We are supporters of responsive disclosure, so if you discover a security vulnerability, we will be waiting to hear you. By reporting security issues you are helping and contributing us.

Steps to a nice report:

Please, would be better for us that you could follow a few steps to reporting us an issue.

Describing the issue

Describe us the vulnerability, type of vulnerability, how can affect us and how we can reproduce it

Media content

You should record a video reproducing the issue and in the future if we have any problem surely we ask you about the "media content" to resolve the problem

Where have you to send the report

To facilitate our issues reading, send us the report to securitygrampus@gmail.com

Examples of accepted vulnerabilities

  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF/XSRF)
  • Remote Code Execution
  • Privilege Escalation
  • Similars....

Not valid vulnerabilities

  • Denial of Service Vulnerabilities
  • False alerts (Honey traps :P)
  • Spam or Social Engineering techniques
